# **App Name**: Liviskov Professional

## Core Features:

- Content Import: Import the existing content from www.liviskov.com.
- Service Showcase: Highlight key services with dedicated sections.
- Testimonial Section: Showcase client testimonials to build trust. This may use a tool that selects relevant excerpts.
- Contact Form: Implement a contact form for inquiries.

## Style Guidelines:

- Primary color: Deep teal (#008080) for a sense of professionalism.
- Background color: Light gray (#F0F0F0) for a clean backdrop.
- Accent color: Muted gold (#B8860B) to add a touch of sophistication.
- Font pairing: 'Playfair' (serif) for headlines and 'PT Sans' (sans-serif) for body text. 'Playfair' has a fashionable feel and is great for headlines, while 'PT Sans' will provide better readability for body text.
- Use modern and minimalist icons related to services.
- Implement a grid-based layout for a structured and clean look.
- Subtle hover effects on service sections and buttons.
